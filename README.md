# 📊 Sales Dashboard (Excel Project)

## 🧾 Project Overview
This project presents an interactive Sales Dashboard created using Microsoft Excel.
It helps analyze business performance by visualizing key metrics like Sales, Profit, Discount, and COGS.

The dashboard enables users to gain insights into:
- Sales trends over time
- Profit distribution across regions
- Segment-wise performance
- Product-level profitability

## 📁 Dataset Description
The dataset contains the following columns:

- Order ID – Unique identifier for each order
- Date – Transaction date
- Month & Year – Time-based analysis
- Region – Sales region (West, East, North, South)
- Segment – Customer type (Consumer, Corporate, Home Office)
- Category & Sub-Category – Product classification
- Product Name – Item sold
- Gross Sale – Total sale before discount
- Discount – Discount applied
- Sales / Net Sale – Final sale after discount
- COGS – Cost of Goods Sold
- Quantity – Units sold
- Profit – Net profit (Sales - COGS)

## 📊 Dashboard Features

### 🔹 KPI Cards
- Total COGS
- Average Sales
- Gross Sales
- Discount
- Net Sales

### 🔹 Visualizations
- Net Sales by Month (Line Chart)
- Profit by Region (Donut Chart)
- Net Sales by Segment (Bar Chart)
- Profit by Products (Column Chart)

### 🔹 Filters (Slicers)
- Year
- Product
- Gross Sales
- Discount
- COGS

## ⚙️ Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers for interactivity
- Data Cleaning & Transformation
- Excel formulas

## 📌 Key Insights
- Consumer segment contributes the highest sales
- Some regions generate higher profits
- Discounts impact overall sales performance
- Product-level analysis helps identify top items
